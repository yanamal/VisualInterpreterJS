This is a small JS library/utility for creating interactive visualizations of interpreting JS (ES5) code.

It is built on top of [JS-Interpreter](https://neil.fraser.name/software/JS-Interpreter/docs.html) (hence ES5).

To use, pull/import `visualize.js` and `visualize.css` into your HTML/JS files.

There are two demos that illustrate the visualizer functionality:

- [demo.html](https://yanamal.github.io/VisualInterpreterJS/demo.html) lets you control each step separately
- [end_to_end.html](https://yanamal.github.io/VisualInterpreterJS/end_to_end.html) is a little more elaborate, and runs several steps on one button click. It also illustrates controlling the DOM via the interpreted code.
